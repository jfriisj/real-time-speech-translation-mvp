## Release Summary
- **Version**: v0.3.0
- **Release Type**: Feature Release (New Service)
- **Environment**: Local Docker Compose
- **Epic Alignment**: Epic 1.5 Ingress Gateway (Phase 1)
- **Key Deliverables**: Python FastAPI Gateway service, WebSocket entry point, Security controls (DoS protection), `AudioInputEvent` production.

## Scope & Changes
- **New Service**: `gateway` service added to docker-compose.
- **Interface**: WebSocket endpoint at `ws://localhost:8000/ws/audio`.
- **Protocol**: Custom handshake with JSON metadata and binary audio chunks.
- **Security**: Origin validation, size limits (1MB), idle timeouts.

## Known Issues / Deferred
- **gRPC Support**: Deferred to Epic 1.5b (post-v0.3.0).
- **Protocol Documentation**: Requires update in `docs/INTERFACES.md`.
